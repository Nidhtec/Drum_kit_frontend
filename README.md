# Drum_kit_frontend
# 🥁 Drum Kit Web App

A simple interactive **Drum Kit web application** built using **HTML, CSS, and JavaScript**.
Users can play different drum sounds by either **clicking the buttons** on the screen or **pressing specific keys on the keyboard**.

This project demonstrates basic **DOM manipulation, event handling, and audio integration using JavaScript**.

---

## 🚀 Features

* Play drum sounds using **keyboard keys**
* Play drum sounds by **clicking buttons**
* **Visual button animation** when a drum is played
* Simple and responsive UI
* Beginner-friendly JavaScript project

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the webpage
* **CSS3** – Styling and layout
* **JavaScript (Vanilla JS)** – Interactivity and sound handling

---

## 🎮 How It Works

Each drum button is associated with a specific keyboard key:

| Key | Drum Sound |
| --- | ---------- |
| W   | Tom 1      |
| A   | Tom 2      |
| S   | Tom 3      |
| D   | Tom 4      |
| J   | Snare      |
| K   | Crash      |
| L   | Kick       |

When the user presses a key or clicks the button:

1. JavaScript detects the event
2. The corresponding **audio file is played**
3. The button shows a **temporary animation effect**

## 📸 Demo

You can interact with the drum kit directly in the browser by clicking the buttons or pressing the corresponding keys on your keyboard.
